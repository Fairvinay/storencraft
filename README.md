## Quick start

**Note:** following instructions are for installing the StoreNCraft [npm package](https://www.npmjs.com/package/@storencraft/storencraft) when integrating StoreNCraft into your own app. To run the repository locally for development, please refer to our [Development Guide](https://docs.storencraft.com/docs/introduction/development).

Use `npm` or `yarn` to install the package.


    npm run build:packages



Check out our [documentation](https://docs.storencraft.com/docs/@storencraft/storencraft/installation) for more details!
